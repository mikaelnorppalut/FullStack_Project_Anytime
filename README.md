# Goal Management MERN App

A full-stack goal management application built with the MERN stack, where users can register, log in, create goals, assign categories, and manage their own goal list.

## Project Information

Created by: Mikael Norppa
Course: Software Development Skills: Full-Stack

## Tech Stack
- MongoDB
- Express
- React
- Node.js
- Redux Toolkit
- JWT authentication

The application runs locally and is not hosted online
